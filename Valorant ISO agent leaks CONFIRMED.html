This agent from China has pretty cool abilities. I'm going to recap them all here.
The Signature Ability for this agent is pretty good. If you activate it, it starts a window of time that's similar to Jett's, who had a nerf on Patch 7.04, which you can read about here. It's much longer though, and if you kill a player during this window of time, you absorb the next bit of damage, including Raze's Ultimate, Showstopper. The C ability conjures a wall similar to Harbour's, but is not breakable by anything, which has INSANE potential on Attack Rounds. The Q ability makes a circle like omen's blind but is faster and smaller and applies Vulnerability insted, causing all damage taken by the player to be doubled for a short duration. The ult gets you and a targeted player in a 1v1. What do YOU think of this agent?
